Distro is a **working music distribution SaaS platform** inspired by services like **DistroKid**.  
It allows artists to submit music, manage metadata, and track approval status through a structured web interface.

This project demonstrates **full-stack web development**, SaaS product thinking, and practical implementation using Laravel technologies.

---

## Features
- Artist music upload and submission
- Metadata management for songs
- Admin review and approval workflow
- Dashboard-style interface
- Responsive and reusable UI components
